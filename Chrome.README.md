## 为何开源？

- 初衷：在和 Tomek 一开始沟通，以及后来形成小册，就是想让更多的人能从中受益，尤其是英语不是那么好的开发者。
- 开源这样的方式可以更好的解决读者遇到的问题，也给后来的人更多参考。
- 开源可以让内容更加丰富，欢迎各位大佬一起来[参与贡献](Contribution.README.md)。
